# 은재 성장 타임라인 모바일 초대장

## 사용 방법
1. `images` 폴더에 은재 사진을 넣습니다.
2. 파일명은 `12.jpg`부터 `23.jpg`까지 맞춰 주세요.
3. `script.js`에서 각 월의 문구와 날짜를 수정합니다.
4. `index.html`에서 생일 날짜·장소·오시는 길 링크를 수정합니다.
5. 로컬에서 `index.html`을 열거나 웹서버에 업로드합니다.

## 핵심 인터랙션
- 각 사진 영역 높이: 모바일 화면의 약 90%
- 다음 사진이 아래쪽에 약 10% 보임
- 세로 스크롤 스냅 적용
- 오른쪽 타임라인 고정
- 현재 보고 있는 월만 선명하게 활성화
- 타임라인 월을 누르면 해당 사진으로 부드럽게 이동

## 사진 권장 비율
- 세로 3:4 또는 4:5
- 인물 얼굴이 화면 중앙보다 살짝 위쪽에 오도록 크롭
- 최소 1200px 이상 권장
